# AI TechBlog

An AI-powered blogging platform built using Django and modern AI technologies.

## Features

- Personalized Recommendation System
- Hybrid Content Recommendation
- Semantic Search
- AI Article Summarization
- AI Toxicity Detection
- RAG Chatbot
- User Dashboard
- Authentication
- Comments
- Bookmarks
- Reading Analytics

## Tech Stack

- Python
- Django
- SQLite
- Sentence Transformers
- Hugging Face Transformers
- PyTorch
- Scikit-learn
- NumPy
- HTML
- CSS
- Bootstrap
- JavaScript

## Installation

```bash
git clone <repo-url>
cd AI-TechBlog
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
